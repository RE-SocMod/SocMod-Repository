# SocMod-Repository
This is a repository used to store the open-source datasets UAV, FSD, and RAC mentioned in the paper "SoCMod: Automating Behavioral Requirements Modeling via
Separation-of-Concerns". We would like to express our gratitude to the authors who created these datasets!
[Note]Based on the research paper scenario, we have made certain adjustments to some of the datasets.

The source of the original dataset and the GitHub project address are as follows:

1. UAV [https://github.com/SAREC-Lab/sUAS-UseCases/tree/SPLC-2020/usecases]
2. FSD/RAC [https://github.com/reg-repo/LATO]
